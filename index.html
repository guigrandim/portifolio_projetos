Nesse projeto utilizando programação em python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como Streamlit e Github foi desenvolvido indicadores de desempenho para analise da empresa Curry Company. Painel Estratégico de Negócios para Companhia de Videogame

Neste projeto, apliquei programação em Python, manipulação de dados e pensamento estratégico para desenvolver uma solução de inteligência de mercado para a BrasCo. Gaming Ltd. Utilizando Streamlit e GitHub, estruturei um dashboard interativo focado na otimização de capital dentro do setor global de games, mapeando KPIs de eficiência. A análise permitiu identificar que 37% do market share concentra-se na América do Norte e que parcerias com a Microsoft oferecem o melhor retorno sobre investimento ($0.32M por ponto). Tambem foram detectadas janelas de oportunidade no 3º ano de vida dos consoles, fornecendo uma base estatística para mitigar riscos e direcionar o lançamento de novos títulos com precisão comercial.
